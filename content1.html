Introduction to JavaScript

JavaScript is a versatile and powerful programming language primarily used for web development. It enables interactive and dynamic content on websites. JavaScript can run in both browsers and servers with the help of frameworks like Node.js.

Here are some fundamental aspects of JavaScript:

  • Dynamic typing
  • First-class functions
  • Prototypal inheritance

Example:


// Simple JavaScript Example
let greeting = "Hello, World!";
console.log(greeting);  // Outputs: Hello, World!

Variables in JavaScript

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, and const.

Declaring Variables:


// Using var, let, and const
var name = "Alice";
let age = 25;
const country = "USA";

Functions in JavaScript

Functions are blocks of code designed to perform a particular task. They are reusable and can be called multiple times in a program.

Example:


function add(a, b) {
    return a + b;
}
console.log(add(5, 3));  // Outputs: 8

Arrays in JavaScript

Arrays are used to store multiple values in a single variable. They can store any type of data, including other arrays.

Example:


let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]);  // Outputs: Apple

Objects in JavaScript

Objects are collections of key-value pairs. They allow you to store data in a structured manner.

Example:


let car = {
    make: "Toyota",
    model: "Camry",
    year: 2021
};
console.log(car.make);  // Outputs: Toyota
